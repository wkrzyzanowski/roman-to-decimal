# Decimal / Roman Number Converter

It is a simple converter between decimal and roman numbers.

# How To Use?

Convert `57` into `LVII`

```sh
python main.py -rom -i 57
```

or

```shell
python main.py --toroman --input 57
```

Convert `XCIV` into `94`

```shell
python main.py -dec -i XCIV
```

or

```shell
python main.py --todecimal --input XCIV
```

# Help

```shell
python main.py --help
```
